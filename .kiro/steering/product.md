# Product Overview

Barcode Attendance System - A web-based student attendance tracking application for schools.

## Core Features
- Barcode scanning (hardware scanners + camera-based via QuaggaJS)
- Student management with automatic barcode generation
- Real-time attendance recording with duplicate prevention
- Parent notifications via SMS (Twilio) and Email (PHPMailer)
- Reporting with CSV, PDF (TCPDF), and Excel (PhpSpreadsheet) exports
- Role-based access control: Admin, Operator, Viewer

## User Roles
- **Admin**: Full system access, user management
- **Operator**: Can scan attendance, manage students
- **Viewer**: Read-only access to reports and records

## Color Scheme
- Primary: Violet (`#8B5CF6`)
- Accent: Orange/Amber for warnings and secondary actions
- UI follows a modern, clean design with Tailwind CSS
