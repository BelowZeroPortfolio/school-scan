# Figma Wireframe Guide - Barcode Attendance System

## Design System

### Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Violet | `#8B5CF6` | Primary buttons, active states, links |
| Violet Light | `#EDE9FE` | Backgrounds, hover states |
| Violet Dark | `#7C3AED` | Button hover, emphasis |
| Orange/Amber | `#F59E0B` | Warnings, secondary actions, premium badges |
| Green | `#10B981` | Success states, present status |
| Red | `#EF4444` | Error states, delete actions, absent status |
| Gray 900 | `#111827` | Primary text |
| Gray 500 | `#6B7280` | Secondary text |
| Gray 100 | `#F3F4F6` | Borders, backgrounds |

### Typography

| Element | Font | Size | Weight |
|---------|------|------|--------|
| Page Title | Inter | 30px (text-3xl) | 600 (semibold) |
| Section Title | Inter | 18px (text-lg) | 600 (semibold) |
| Body Text | Inter | 14px (text-sm) | 400 (normal) |
| Labels | Inter | 14px (text-sm) | 500 (medium) |
| Small Text | Inter | 12px (text-xs) | 500 (medium) |

### Spacing Scale

- `4px` - Tight spacing (gap-1)
- `8px` - Small spacing (gap-2, p-2)
- `12px` - Medium spacing (gap-3, p-3)
- `16px` - Default spacing (gap-4, p-4)
- `24px` - Large spacing (gap-6, p-6)
- `32px` - Section spacing (py-8)


### Border Radius

- `8px` (rounded-lg) - Buttons, inputs
- `12px` (rounded-xl) - Cards, modals
- `16px` (rounded-2xl) - Large cards
- `9999px` (rounded-full) - Badges, avatars

## Responsive Breakpoints

| Breakpoint | Width | Layout |
|------------|-------|--------|
| Mobile | < 768px | Single column, hamburger menu |
| Tablet | 768px - 1024px | Collapsible sidebar |
| Desktop | > 1024px | Full sidebar, multi-column |

### Mobile Sidebar Behavior
- Hidden by default
- Slide-in from left on hamburger click
- Overlay with backdrop
- Close on outside click or X button

### Desktop Sidebar Behavior
- Collapsible (64px collapsed, 256px expanded)
- Tooltips shown when collapsed
- Persists collapse state

## Figma Implementation Tips

1. **Use Auto Layout** for all components
2. **Create Components** for buttons, inputs, cards, badges
3. **Use Variables** for colors and spacing
4. **Create Variants** for button states (default, hover, disabled)
5. **Use Frames** with proper constraints for responsive design
6. **Name layers** consistently (e.g., `btn/primary`, `card/default`)

### Recommended Figma Plugins
- Iconify (for icons)
- Tailwind CSS Color Generator
- Content Reel (for placeholder data)
